# 福彩3D 预测系统 v1.0

基于 **中国福利彩票官方数据** + **9 种统计策略** + **15 维技术指标** 的号码分析工具。

⚠ **彩票开奖是独立随机事件，所有输出仅供娱乐，不构成投注建议。请理性购彩，量力而行。**

---

## 开机自启（一次性配置）

**Windows：双击 `setup_autostart.bat` 即可完成配置。** 之后每次开机系统会自动在后台启动预测调度器，无需手动操作。

如果上面方法失败，手动操作：
1. 右键 `start.vbs` → 创建快捷方式
2. `Win+R` → 输入 `shell:startup` → 回车
3. 把快捷方式拖进去

完成后，**重启电脑就会自动运行**，不需要再手动启动。

## 快速开始

### 1. 环境要求

- Python 3.8+
- pip

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 运行

```bash
# 手动预测一次（预测 + 推送飞书，执行完就退出）
python main.py predict

# 全自动值守（推荐日常使用，启动后一直在后台跑）
python main.py schedule

# 仅看统计（不预测、不推送，只打印到屏幕）
python main.py stats

# 手动触发周报
python main.py weekly

# 强制刷新数据（从官网重新抓取全部历史）
python main.py refresh

# 预测但不推送飞书（测试用）
python main.py predict --no-push
```

## 三种运行方式详解

| 模式 | 命令 | 什么时候用 |
|------|------|-----------|

## 预测原理

本系统基于 **数学确定性**，不是随机猜测：

- **数据源**：中国福利彩票网 (cwl.gov.cn) 官方 API
- **随机种子**：以预测期号作为固定种子，确保同一期永远输出相同结果
- **15 维指标**：频率、遗漏、和值、跨度、012 路、AC 值、奇偶、大小、质合、形态、邻孤传、复隔中、热温冷、位差位和、振幅
- **9 种策略**：
  - 规则策略：热号加权 / 冷号回补 / 和值回归 / 形态跟随 / 遗漏组合 / 纯随机基线
  - 统计策略：加权马尔可夫链 / 多指标综合评分 / 移动平均趋势
- **集成投票**：频率投票 + 位选投票 → 共识推荐

**确定性保证**：相同数据 + 相同配置 + 相同期号 → 永远相同结果。任何人、任何机器运行，同期预测完全一致。

## 注意事项

1. **`data/history.json` 必须保留** — 删除后重新抓取可能导致预测结果变化
2. **不要修改 `config.yaml` 中的 `samples` 值** — 会影响随机序列，改变预测结果
3. 飞书 Webhook 地址是敏感信息，不要泄露给他人
4. 每天 21:15 开奖，20:50 截止投注
5. 节假日休市期间系统不会预测

## 常见问题

**Q: 为什么同一期预测永远一样？**
系统用预测期号（如 2026155）作为 Python `random.seed()` 的固定种子，Mersenne Twister 算法保证给定种子永远产生相同随机序列。

**Q: 别人运行结果会和我一样吗？**
只要 `data/history.json` 和 `config.yaml` 完全相同，结果就完全一样。

**Q: 数据会自动更新吗？**
`schedule` 模式下会自动检测并追加新一期数据。手动模式需要 `python main.py refresh`。

**Q: 预测准吗？**
彩票开奖是独立随机事件，任何策略都无法超越随机概率。本系统仅供娱乐和数据研究。
