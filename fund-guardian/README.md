# 📊 基金守护者 (Fund Guardian)

全天候基金量化分析 + AI 政策解读 + 飞书推送

## 六层策略架构

| 层 | 名称 | 做什么 |
|----|------|--------|
| ① | 四季分仓 | 判断市场季节，动态调整仓位（春90%/夏70%/秋40%/冬20%） |
| ② | 分批进出 | 买卖分三批执行，降低一次性踩错风险 |
| ③ | 情绪温度 | 反向指标——极度贪婪时减仓，极度恐惧时加仓 |
| ④ | 行业轮动 | 追踪28个申万行业动量，超跌加分、过热减分 |
| ⑤ | 止损止盈 | 硬止损-8%、主动止盈+12%、回撤止损5% |
| ⑥ | 相关性风控 | 检测持仓间高相关性，防止"假分散" |

## 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 配置（复制 .env.example 为 .env，填入你的 API Key）
cp .env.example .env

# 3. 测试运行（单次）
python main.py --once

# 4. 持续监控（24小时）
python main.py
```

## 日常工作流

```
交易日 09:15 → 📡 早盘新闻扫描（重大事件即时推送）
交易日 12:00 → 📡 午间更新
交易日 14:30 → 📡 尾盘监控（大盘异动推送）
交易日 15:30 → 📊 收盘快报（实时净值+策略评分）
交易日 21:00 → 📈 日终完整分析 + 飞书日报推送
每周日 20:00 → 🔄 策略回测 + 参数自优化
```

## 配置关注基金

编辑 `config.py` 中的 `WATCH_FUNDS` 字典：

```python
WATCH_FUNDS = {
    "你起的名字": "基金6位代码",  # 在支付宝/天天基金查看
}
```

## 获取飞书 Webhook

1. 打开飞书 → 任一群聊 → 设置 → 群机器人
2. 添加自定义机器人 → 复制 Webhook URL
3. 填入 `.env` 的 `FEISHU_WEBHOOK_URL`

## 获取 Claude API Key

1. 访问 https://console.anthropic.com
2. 创建 API Key
3. 填入 `.env` 的 `ANTHROPIC_API_KEY`

> AI 分析是可选的。没有 API Key 也能跑，只是不做新闻深度解读。
